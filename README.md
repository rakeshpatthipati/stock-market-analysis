# Stock Market Analysis Dashboard

A complete data science project analyzing AAPL, Tesla and Infosys stock performance for 2024.

**Author:** Rakesh  
**Date:** April 2026  
**Tools:** Python, JupyterLab, yfinance, pandas, matplotlib

## What This Project Does
- Fetches 1 year of real stock price data from Yahoo Finance
- Computes daily returns, moving averages and volatility
- Generates 6 professional charts comparing all 3 stocks
- Identifies the best performing stock of 2024

## Files
- `Stock Analysis.ipynb` — Main Jupyter Notebook
- `stock_analysis.png` — 6-panel visualization chart

## How to Run
pip install jupyter pandas yfinance matplotlib seaborn
jupyter notebook
